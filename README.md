# grlc-mini-demo
Mini demo of a website built using a grlc-API.

This repo contains 3 SPARQL queries:

 - genres.sparql: List music genres from DBpedia
 - bands.sparql: List bands in a given genre
 - bandinfo.sparql: List band information

These queries are converted to an API using [grlc](https://github.com/CLARIAH/grlc):

https://grlc.io/api-git/c-martinez/grlc-mini-demo

A web interface (static website) is used to query and display results from this API directly in your browser, without needing to know SPARQL.

Visit https://c-martinez.github.io/grlc-mini-demo/music_explorer.html
